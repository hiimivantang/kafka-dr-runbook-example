You can switch over to https://zookeeper.apache.org/doc/r3.2.2/zookeeperHierarchicalQuorums.html[Zookeeper hierarchical quorum] by updating `zookeeper.properties` for each zookeeper node and restarting Zookeeper service.

See following snippet of `zookeeper.properties` containing configuration for a hierarchical quorum with 3+3 Zookeeper nodes across two DCs.

[source,properties]
----
server.1=<ZK1_HOSTNAME>:2888:3888
server.2=<ZK2_HOSTNAME>:2888:3888
server.3=<ZK3_HOSTNAME>:2888:3888
server.4=<ZK4_HOSTNAME>:2888:3888
server.5=<ZK5_HOSTNAME>:2888:3888
server.6=<ZK6_HOSTNAME>:2888:3888

group.1=1:2:3
group.2=4:5:6

weight.1=1
weight.2=1
weight.3=1
weight.4=1
weight.5=1
weight.6=1
----
